### Syntaxes

- R Extended Syntax
- Rcpp Syntax
- R Sweave and R Markdown syntaxes


### Build commands

Build commands <kbd>C</kbd>+<kbd>b</kbd> for R Package, Rmarkdown and Rnw files.
